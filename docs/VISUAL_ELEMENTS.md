# 🎨 Auto-Fix Bot 視覺元素指南

## 視覺設計理念

Auto-Fix Bot 的視覺元素設計旨在突出**效率**、**速度**和**專業性**。

## 🎯 品牌色彩

### 主色調
```css
--primary-color: #00D9FF;      /* 科技藍 - 代表創新和效率 */
--secondary-color: #7B68EE;    /* 活力紫 - 代表智能和力量 */
--accent-color: #00FF7F;       /* 成功綠 - 代表完成和正確 */
--warning-color: #FFD700;      /* 警告金 - 代表注意事項 */
--error-color: #FF6B6B;        /* 錯誤紅 - 代表需要修復 */
```

### 輔助色彩
```css
--dark-bg: #1E1E1E;            /* 深色背景 */
--light-bg: #F5F5F5;           /* 淺色背景 */

--text-secondary: #B0B0B0;     /* 次要文字 */
```
