📂 目錄結構：
├── dist/            ⭐ 這是生產構建（建議直接使用）
│   ├── index.html   → 在瀏覽器中打開此文件以預覽網站
│   ├── assets/      → 打包的 CSS、JS 和其他資源文件
│   └── ...
├── src/             📝 原始碼目錄（用於開發和修改）
│   ├── components/  → React 組件
│   ├── App.jsx      → 主應用組件
│   ├── index.js     → 入口文件
│   └── ...
├── package.json     📦 專案依賴配置
├── README.md        📖 專案文檔
└── ...

🚀 快速開始（兩種選擇）：
選擇 1：直接使用（建議初學者）
1. 進入 dist/ 目錄
2. 雙擊 index.html 文件
3. 網站將在您的瀏覽器中打開！

選擇 2：開發模式（適合有編碼經驗的用戶）
1. 確保已安裝 Node.js（推薦版本：16+）
2. 在專案根目錄中打開終端/命令行
3. 執行：npm install
4. 執行：npm start
5. 開發伺服器將自動在您的瀏覽器中打開

💡 提示：
- dist/ 目錄包含優化的構建，可以直接部署到伺服器
- 根目錄的 index.html 不是最終版本 — 請始終使用 dist/ 中的版本
- 要修改代碼，請編輯 src/ 中的文件並重新構建專案
- 專案使用 TypeScript 和 Tailwind CSS 技術棧
